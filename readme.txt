== Description ==

Display VKontakte donate widget on your site



== Installation ==

How to install the widget and get it working.


* 1) Copy folder with this widget to wp-content/plugins directory
* 2) Activate VKDonate
* 3) Put VKDonate widget on sidebar (Apperance -> Widgets) 
* 4) Save your VKontakte API ID
* 5) Enjoy, dudes =)

== Frequently Asked Questions ==

[English] What is the option VKontakte API ID 
Perform the login to your Vkontakte account. Go to the "Connecting the widget on your site" page and copy your number of VKontakte API ID, for example: apiId: 1960175
                
[Russian] Что такое опция VKontakte API ID
Выполните логин в ваш аккаунт Vkontakte. Перейдите на страницу "Подключение виджета к Вашему сайту". В поле "Код для вставки:" найдите и скопируйте цифровую часть VKontakte API ID, для примера: apiId: 1960175
